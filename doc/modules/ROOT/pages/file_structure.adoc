[#structure]
= Directory Structure
:idprefix: structure_

The entire library can be consumed via `<boost/decimal.hpp>`, or by independently selecting any of the library headers.
Below is the directory structure for reference:

----
boost/
├── decimal.hpp
└── decimal/
    ├── bid_conversion.hpp
    ├── cfloat.hpp
    ├── charconv.hpp
    ├── cmath.hpp
    ├── cstdio.hpp
    ├── cstdlib.hpp
    ├── decimal32_t.hpp
    ├── decimal64_t.hpp
    ├── decimal128_t.hpp
    ├── decimal_fast32_t.hpp
    ├── decimal_fast64_t.hpp
    ├── decimal_fast128_h.hpp
    ├── dpd_conversions.hpp
    ├── fenv.hpp
    ├── fmt_format.hpp
    ├── format.hpp
    ├── fwd.hpp
    ├── hash.hpp
    ├── iostream.hpp
    ├── literals.hpp
    ├── numbers.hpp
    └── string.hpp
----
